# Known Limitations

## Current limitations

- The project still needs local validation in Godot.
- No CI validation is configured yet.
- Export presets are not validated yet.
- Demo scene maturity needs confirmation.
- Advanced settings menu is not included.
- Save system is not included.
- Inventory system is not included.
- Enemy AI is intentionally not included.
- External asset packs are intentionally not bundled.

## Why some systems are missing

The template is designed to stay small and beginner-friendly.

Complex systems should be added only after the base demo is stable.

## Recommended next additions

- validated micro horror demo flow;
- optional atmosphere effect example;
- export preset examples;
- CI check for project structure;
- settings menu later.
