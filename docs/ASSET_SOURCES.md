# Asset Sources

This page helps template users find third-party assets safely.

## Recommended sources

| Source | Useful for | Notes |
|---|---|---|
| Godot Asset Library | addons, tools, shaders, examples | Prefer clear licenses and active projects. |
| GitHub | open-source plugins and templates | Read the LICENSE file before copying code. |
| OpenGameArt | art, music, sound effects | Check the license on each asset page. |
| Kenney | placeholder assets | Good for prototypes and game jams. |
| Freesound | ambience and sound effects | Licenses differ per sound. |
| itch.io asset packs | low-poly, PSX, horror packs | Check commercial use and redistribution terms. |
| Poly Haven | HDRIs, textures, models | Good for environment work. |
| ambientCG | PBR textures | Good for walls, floors and props. |

## Project rule

Before adding any external asset, update:

- `docs/CREDITS.md`
- `docs/LICENSE_CHECKLIST.md`

If the license is unclear, do not add the asset to the template.
