# Prompt 03 - Add Horror Event

Task:

Add one small scripted horror event to the demo level.

Good event examples:

- light flicker;
- short distant sound;
- text appears on a wall;
- door closes behind the player;
- objective text changes after entering an area.

Rules:

- Event must be one-time unless requested otherwise.
- Event must not require enemy AI.
- Event must not kill the player.
- Event must not block progress forever.
- Keep intensity low for template examples.

Definition of Done:

- Event is easy to find and test.
- Event uses clear scene/script names.
- `docs/CHANGELOG.md` is updated.
