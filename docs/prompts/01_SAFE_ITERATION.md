# Prompt 01 - Safe Iteration

You are working in a Godot atmospheric horror template.

Goal:

Implement one small improvement while keeping the project runnable.

Rules:

- Read `AGENTS.md` before editing.
- Touch the smallest possible set of files.
- Do not change unrelated systems.
- Do not add external assets.
- Update `docs/CHANGELOG.md`.
- If a design decision is made, update `docs/DECISIONS.md`.

Definition of Done:

- The project opens in Godot.
- `game/scenes/Main.tscn` still exists.
- No autoload paths are broken.
- The change can be tested manually.
- Documentation is updated.

Final response format:

```md
## Summary

## Files changed

## Manual test

## Notes
```
