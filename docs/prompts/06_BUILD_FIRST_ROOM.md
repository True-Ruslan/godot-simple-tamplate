# Prompt 06 - Build First Room

## Task

Build or modify the first playable room for the current game.

## Requirements

- Keep the scene small.
- Use placeholder geometry if needed.
- Include one entry point.
- Include one objective object.
- Include one interaction object.
- Include one exit or return path.
- Do not add external assets.

## Suggested room types

- classroom;
- apartment kitchen;
- motel room;
- gas station store;
- tower cabin;
- office room.

## Manual test

1. Run the main scene.
2. Confirm the player spawns correctly.
3. Walk around the room.
4. Interact with the required object.
5. Confirm there is no softlock.
