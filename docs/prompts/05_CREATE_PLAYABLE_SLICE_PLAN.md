# Prompt 05 - Create Playable Slice Plan

## Task

Create a plan for the smallest playable slice of the current game.

## Read first

- `AGENTS.md`
- `docs/CONTEXT.md`
- `docs/SCENARIO.md` if it exists
- `docs/examples/PLAYABLE_SLICE_TEMPLATE.md` if it exists

## Output required

Create or update a document called:

```text
docs/PLAYABLE_SLICE.md
```

The document must include:

- working title;
- duration target;
- location;
- start state;
- end state;
- objective chain;
- required objects;
- required mechanics;
- one horror event;
- manual test checklist;
- out-of-scope list.

## Scope rule

The playable slice must be completable in 5-15 minutes.

## Do not

- add enemy AI;
- design a full map;
- add multiple endings;
- require more than one new mechanic.
