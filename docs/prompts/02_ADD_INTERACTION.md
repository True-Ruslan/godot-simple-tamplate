# Prompt 02 - Add Interaction

Task:

Add one simple interaction to the demo level.

Examples:

- readable note;
- door toggle;
- light switch;
- objective update;
- one-time sound trigger.

Requirements:

- Use the existing interaction style.
- Keep the code beginner-friendly.
- Do not add new dependencies.
- Update `docs/CHANGELOG.md`.

Manual test:

1. Run the main scene.
2. Walk to the object.
3. Confirm that the interaction prompt appears.
4. Press the interaction key.
5. Confirm that the action runs once or toggles correctly.
