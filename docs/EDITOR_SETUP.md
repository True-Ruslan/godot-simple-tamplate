# Editor Setup

Recommended setup for working with this template.

## Godot

Use Godot 4.7 or newer unless the project support matrix says otherwise.

## Editor settings

Recommended:

- enable visible collision shapes during debugging;
- keep the FileSystem dock visible;
- use the Remote scene tree when testing;
- check the Output panel after every run.

## Useful panels

- Scene tree;
- Inspector;
- FileSystem;
- Output;
- Debugger;
- Remote scene tree.

## Common checks

- Does `project.godot` show the correct main scene?
- Do autoload scripts load without errors?
- Does the player scene open?
- Does the demo level open?
- Are interactable objects assigned scripts?
