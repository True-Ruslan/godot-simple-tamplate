# Template Ready Definition

The template is ready for public reuse when these conditions are true.

## Required

- Project opens in Godot.
- Main scene runs.
- Basic controls work.
- Basic interaction works.
- Documentation has a clear start point.
- License and credits are clear.
- Known limitations are documented.

## Nice to have

- One complete micro horror demo.
- Screenshots.
- CI checks.
- Export preset examples.

Validate runtime behavior locally.
