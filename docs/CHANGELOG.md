# CHANGELOG.md

Формат основан на Keep a Changelog.

## [0.1.0] — 2026-07-07

### Added

- Создан Godot-проект.
- Добавлена основная сцена `Main.tscn`.
- Добавлен демо-уровень `TemplateLevel.tscn`.
- Добавлен first-person player controller.
- Добавлен фонарик с зарядом.
- Добавлена interaction raycast-система.
- Добавлена интерактивная дверь.
- Добавлена интерактивная записка.
- Добавлен выключатель света.
- Добавлен HUD.
- Добавлен `ObjectiveManager`.
- Добавлен `HorrorEventManager`.
- Добавлен `GameState`.
- Добавлена документация для человека и ИИ-агента.
- Добавлен первый видимый scripted event: после чтения записки появляется надпись на доске.
- Добавлен минимальный финальный шаг демо: дверь открывается после события и показывает завершающее сообщение.
- Добавлены стартовое меню, меню паузы и базовое меню настроек.
- Добавлены showcase-примеры для audio, speech, dialogue и postfx.
- Добавлены документы для интеграций и реестр сторонних ресурсов.

### Fixed

- Исправлена обработка mouse look: поворот камеры теперь обрабатывается до UI, а HUD не блокирует движение мыши.
