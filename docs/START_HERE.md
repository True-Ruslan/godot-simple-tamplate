# Start Here

This is the recommended entry point for users who create a new game from this template.

## Fast path

1. Open the project in Godot.
2. Run `game/scenes/Main.tscn`.
3. Read `docs/TEMPLATE_USAGE.md`.
4. Choose one blueprint from `docs/game_blueprints/`.
5. Create `docs/SCENARIO.md` from `docs/SCENARIO_TEMPLATE.md`.
6. Create `docs/PLAYABLE_SLICE.md` from `docs/examples/PLAYABLE_SLICE_TEMPLATE.md`.
7. Build one room.
8. Add one interaction.
9. Add one small horror event.
10. Playtest.

## Best first documents

- `README.md`
- `docs/DOCUMENTATION_INDEX.md`
- `docs/TEMPLATE_USAGE.md`
- `docs/examples/SCHOOL_HORROR_EXAMPLE.md`
- `docs/PLAYTEST_CHECKLIST.md`

## Best first prompt for an AI agent

Start with:

```text
docs/prompts/00_REPOSITORY_REVIEW.md
```

Then use:

```text
docs/prompts/04_CONVERT_TEMPLATE_TO_GAME.md
```

## Main rule

Do not build the whole game at once. The best result is one small playable slice that always runs.
