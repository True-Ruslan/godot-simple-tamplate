# Agent Limits

## Agents may do

- Improve documentation.
- Add small scripts.
- Add one interaction at a time.
- Add one small event at a time.
- Fix broken references.
- Update changelog and task files.

## Ask before

- Replacing the player controller.
- Changing folder structure.
- Adding large assets.
- Adding new dependencies.
- Adding multiplayer.
- Adding complex AI.

## Best task size

A good task changes 1-5 files and can be tested in less than 5 minutes.
