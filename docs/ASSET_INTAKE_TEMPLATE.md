# Asset Intake Template

Copy this block into `docs/CREDITS.md` whenever you add an external asset.

```md
## Asset Name

- Type: model / texture / sound / music / font / shader / addon / code
- Author:
- Source URL:
- License:
- Commercial use: yes / no / unknown
- Modification allowed: yes / no / unknown
- Attribution required: yes / no / unknown
- Added in commit / PR:
- Notes:
```

## Review checklist

- [ ] Source URL is saved.
- [ ] Author is saved.
- [ ] License is saved.
- [ ] License allows the intended use.
- [ ] Attribution text is ready if required.
- [ ] Asset is not larger than needed.
- [ ] Demo scene still runs.
