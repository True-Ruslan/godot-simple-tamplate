# Playtest Checklist

Use this checklist for quick manual playtests.

## Setup

- [ ] Correct Godot version is installed.
- [ ] Project opens without missing script errors.
- [ ] Main scene starts.

## Controls

- [ ] Mouse look works.
- [ ] WASD movement works.
- [ ] Sprint works.
- [ ] Crouch works.
- [ ] Interaction key works.
- [ ] Flashlight key works.
- [ ] Escape releases or handles mouse capture.

## Interaction

- [ ] Interaction prompt appears near interactable objects.
- [ ] Door interaction works.
- [ ] Note interaction works.
- [ ] Light switch interaction works.
- [ ] Objective text updates when expected.

## Feel

- [ ] Movement speed feels acceptable.
- [ ] Flashlight is useful but not overpowered.
- [ ] UI text is readable.
- [ ] The demo has a clear start and end.
- [ ] No confusing dead ends exist in the demo.
