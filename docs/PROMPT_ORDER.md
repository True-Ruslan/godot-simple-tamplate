# Prompt Order

Recommended order for AI-agent prompts.

## For repository review

1. `docs/prompts/00_REPOSITORY_REVIEW.md`
2. `docs/AGENT_NEXT_PROMPT.md`

## For creating a new game

1. `docs/prompts/04_CONVERT_TEMPLATE_TO_GAME.md`
2. `docs/prompts/05_CREATE_PLAYABLE_SLICE_PLAN.md`
3. `docs/prompts/06_BUILD_FIRST_ROOM.md`
4. `docs/prompts/07_REPLACE_DEMO_SAFELY.md`

## For adding features

1. `docs/prompts/01_SAFE_ITERATION.md`
2. `docs/prompts/02_ADD_INTERACTION.md`
3. `docs/prompts/03_ADD_HORROR_EVENT.md`

## Rule

Run only one prompt at a time and verify the project after each result.
