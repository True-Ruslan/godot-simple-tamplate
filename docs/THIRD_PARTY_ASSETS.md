# Third-party Assets

Use this file to record every external asset, addon, shader, sound, font, texture, model, music track or code snippet.

## Current bundled external assets

None.

The current showcase uses built-in Godot nodes, scripts and generated placeholder behavior.

## Registry template

| Asset | Type | Source | Author | License | Used in | Commercial safe | Notes |
|---|---|---|---|---|---|---|---|
| Example asset | sound / texture / model / font / shader / addon | URL | Author | CC0 / MIT / CC-BY | Scene or script | yes / no / unknown | Notes |

## Recommended policy

1. Prefer CC0 for art, textures, sound and music.
2. Prefer MIT, BSD or Apache-2.0 for code and addons.
3. Avoid non-commercial licenses.
4. Avoid assets without a clear source URL.
5. Do not commit large assets until they are needed by a showcase.

## Required before commit

- Source URL is saved.
- Author is saved.
- License is saved.
- Commercial use status is clear.
- Attribution text is ready when required.
