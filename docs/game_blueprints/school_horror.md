# School Horror Blueprint

## Core fantasy

The player returns to an old school at night and uncovers a buried memory.

## Recommended locations

- entrance hall;
- classroom;
- teacher room;
- corridor;
- toilet;
- gym;
- archive;
- basement.

## Recommended mechanics

- flashlight;
- locked classrooms;
- blackboard messages;
- school bell events;
- notes and old journals;
- power switch puzzle.

## Example objectives

1. Find the teacher room key.
2. Restore power on the first floor.
3. Read the old class journal.
4. Find chalk.
5. Return to the classroom.

## Example horror events

- A phrase appears on the blackboard.
- A school bell rings without visible cause.
- Footsteps are heard from an upper floor.
- A classroom door closes behind the player.
- The objective text changes after reading a note.

## MVP scope

One floor, three rooms, one corridor, one note, one key, one blackboard event.
