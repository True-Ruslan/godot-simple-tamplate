# Roadside Horror Blueprint

## Core fantasy

The player stops during a late-night trip and slowly realizes that the place is not as empty as it looks.

## Recommended locations

- car;
- roadside;
- small store;
- gas station;
- toilet building;
- back room;
- forest edge.

## Recommended mechanics

- car start / stop state;
- flashlight;
- locked back room;
- cash register or note interaction;
- distant audio cues;
- objective-based exploration.

## Example objectives

1. Check the car.
2. Find someone at the store.
3. Restore power.
4. Find the restroom key.
5. Return to the car.

## Example horror events

- Car headlights flicker.
- Store radio plays distorted speech.
- A light turns on in a locked room.
- The car is no longer where it was parked.
- A sound comes from the forest edge.

## MVP scope

One exterior area, one store interior, one locked room, one car objective.
