# Forest Lookout Horror Blueprint

## Core fantasy

The player works alone in an isolated lookout tower and notices strange changes in the surrounding forest.

## Recommended locations

- tower cabin;
- stairs;
- radio desk;
- supply shed;
- forest path;
- locked gate;
- observation point.

## Recommended mechanics

- radio messages;
- binocular or camera interaction;
- flashlight;
- supply checklist;
- distant lights;
- weather and ambience changes.

## Example objectives

1. Start the shift.
2. Check the radio.
3. Inspect the supply shed.
4. Report a distant light.
5. Return to the tower.

## Example horror events

- Radio receives a message that should not exist.
- A light appears between trees.
- The supply shed door is open after being locked.
- Footsteps are heard under the tower.
- The forest sound suddenly stops.

## MVP scope

One tower room, one exterior path, one shed, one radio event.
