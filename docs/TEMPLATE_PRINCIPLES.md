# Template Principles

These principles should guide future changes.

## 1. Runnable first

The template must stay runnable after every meaningful change.

## 2. Small scope

Prefer one complete small example over many incomplete systems.

## 3. Beginner-friendly

Code, scenes and docs should be understandable for a beginner.

## 4. Agent-friendly

AI agents need clear rules, small tasks and explicit definitions of done.

## 5. Legally safe

Do not bundle assets or code with unclear licenses.

## 6. Horror through atmosphere

The base template should support tension, exploration, light, sound and simple events before enemies or combat.

## 7. Extensible, not heavy

The template should be easy to extend, but not overloaded with systems.
