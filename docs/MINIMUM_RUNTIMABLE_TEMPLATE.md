# Minimum Runnable Template

A minimum runnable template must include:

- `project.godot`;
- valid main scene;
- player scene;
- one level scene;
- one interaction example;
- no missing autoload scripts;
- no missing required resources.

## Minimum player loop

1. Start scene.
2. Move.
3. Look around.
4. Interact.
5. Receive objective feedback.
6. Exit.

## Minimum documentation loop

1. Read README.
2. Read `docs/START_HERE.md`.
3. Run the project.
4. Use one prompt.
5. Make one small change.
6. Playtest.

## Release rule

If the minimum player loop does not work, do not release.
