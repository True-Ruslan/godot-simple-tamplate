# Repository Settings

Recommended GitHub settings for this template.

## General

- Enable template repository mode.
- Add topics: `godot`, `godot4`, `horror`, `template`, `gamedev`, `gdscript`.
- Add a short description about first-person atmospheric horror prototypes.

## Pull requests

- Prefer squash merge for documentation and small changes.
- Keep PRs focused.
- Review scene path changes carefully.

## Branches

Recommended default branch: `main`.

Optional branch protection:

- require PR before merge;
- require linear history if the project grows;
- require status checks after CI is added.

## Releases

Use releases for stable template snapshots:

- `v0.1.0` initial runnable template;
- `v0.2.0` documentation and blueprint pack;
- `v0.3.0` improved demo and examples.
