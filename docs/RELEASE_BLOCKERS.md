# Release Blockers

Track items that block the first release.

## Blocking for v0.1.0

- [ ] Project must be opened in Godot locally.
- [ ] Main scene must run.
- [ ] Basic controls must be tested.
- [ ] Interaction loop must be tested.
- [ ] Credits and license files must be checked.
- [ ] README clone URL must be reviewed after final repository rename.

## Not blocking

- CI setup.
- Export presets.
- Save system.
- Settings menu.
- Advanced visual effects.
- Final art assets.

## Decision

Do not tag `v0.1.0` until the blocking items are complete.
