# Integrations

This folder describes focused integration examples for the template.

## Goal

Show common Godot systems in small isolated examples.

## Built-in examples

- Audio events.
- Text to speech narration.
- Simple dialogue runner.
- Post effects controller.
- Third-party asset registry.

## Rules

- Keep each example small.
- Prefer safe licenses.
- Record external resources in `docs/THIRD_PARTY_ASSETS.md`.
- Do not add large binary files without review.

## Scene location

```text
game/scenes/examples/
```

## Script location

```text
game/scripts/audio/
game/scripts/dialogue/
game/scripts/postfx/
```
