# Optional Addons

The base template works without addons.

## Intake checklist

- [ ] License is clear.
- [ ] Version is recorded.
- [ ] Source URL is recorded.
- [ ] Only required files are committed.
- [ ] `docs/THIRD_PARTY_ASSETS.md` is updated.
- [ ] Project opens after enabling the addon.

## Rule

An optional addon should not become required for opening the template unless the decision is documented in `docs/DECISIONS.md`.
