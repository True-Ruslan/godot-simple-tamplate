# School Horror Example

Working title: `The Chalk`

## Pitch

A former student returns to an old school at night to collect archive documents. The school is empty, but old classroom memories begin to appear as physical events.

## Target duration

10-15 minutes for the first playable slice.

## Location

One school corridor and one classroom.

## Player goal

Find the old class journal and leave the school.

## Required objects

- classroom door;
- readable note;
- blackboard;
- light switch;
- flashlight;
- key or simple locked state.

## Objective chain

1. Enter the corridor.
2. Find the classroom.
3. Read the note on the desk.
4. Turn on the light.
5. Look at the blackboard.
6. Trigger the chalk message event.
7. Pick up the journal.
8. Return to the exit.

## Horror event

When the player reads the note and turns around, the classroom light flickers. A short chalk sound plays. A message appears on the blackboard.

Suggested message:

```text
You are late again.
```

## Scope limit

Do not add:

- enemy AI;
- chase sequence;
- inventory grid;
- multiple floors;
- more than one ending;
- more than three notes.

## Done

The slice is done when a player can start, find the classroom, read the note, see the event, pick up the journal and return to the exit.
