# Micro Objective Chains

Small objective chains help keep short horror games focused.

## Chain 1 - Read and react

1. Enter the room.
2. Read a note.
3. Hear a sound.
4. Check the changed object.
5. Leave the room.

## Chain 2 - Restore power

1. Find a dark room.
2. Locate the fuse box.
3. Turn power on.
4. Notice that something changed.
5. Continue through the unlocked door.

## Chain 3 - Key and door

1. Find a locked door.
2. Search the nearby room.
3. Pick up a key.
4. Hear a distant event.
5. Unlock the door.

## Chain 4 - Return route changes

1. Enter a safe location.
2. Complete a simple task.
3. Return to the previous corridor.
4. See that one object changed.
5. Follow the new objective.

## Chain 5 - Audio lead

1. Hear a repeated sound.
2. Follow the sound.
3. Find a readable object.
4. Trigger a quiet event.
5. Objective updates.
