# Owner Actions

Actions that require repository owner attention.

## Repository settings

- Enable template repository mode.
- Confirm final repository name.
- Add repository description.
- Add repository topics.

## Local validation

- Clone the repository.
- Open in Godot 4.7 or newer.
- Run the main scene.
- Fill `docs/VALIDATION_REPORT_TEMPLATE.md`.

## Release

- Review `docs/RELEASE_NOTES_v0.1.0.md`.
- Complete `docs/RELEASE_CHECKLIST.md`.
- Create `v0.1.0` tag only after validation.
