## Project positioning

This is not a general-purpose Godot starter and not a shooter framework.

It is focused on:

- short atmospheric horror;
- first-person exploration;
- simple interaction;
- notes, doors, lights and objectives;
- small playable slices;
- AI-assisted iterative development.

## What should stay true

- A new user can clone the project and run a demo.
- A beginner can understand the structure.
- An AI agent has clear constraints.
- Third-party assets are handled carefully.
- The template stays small and reusable.
