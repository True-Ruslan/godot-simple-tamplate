# Local Test Commands

Godot validation is mostly editor-based, but these commands can help.

## Open project from terminal

```bash
godot --path .
```

## Run project

```bash
godot --path .
```

Then press Play in the editor if needed.

## Export later

Export commands should be added only after export presets are created and validated.

## Notes

Command names may differ depending on how Godot is installed. On some systems the executable may be named:

```text
godot4
Godot_v4.7-stable
```
