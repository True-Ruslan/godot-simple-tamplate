# Roadmap

This roadmap describes the intended evolution of the template.

## v0.1 - Runnable base template

- First-person player.
- Basic demo level.
- Interaction raycast.
- Door, note and light switch examples.
- Flashlight.
- Objective manager.
- Horror event manager.
- Basic documentation.

## v0.2 - Documentation and workflow pack

- Asset source guide.
- Addon guide.
- License checklist.
- Agent workflow.
- Prompt pack.
- Game blueprints.
- Repository maintenance docs.

## v0.3 - Better starter game slice

- More polished demo room.
- One complete micro objective chain.
- One scripted horror event.
- Improved HUD messages.
- Example scenario document.
- Manual playtest checklist.

## v0.4 - Atmosphere pack

- Optional VHS or film grain shader example.
- Audio event examples.
- Light flicker example.
- Ambience bus setup.
- Horror pacing notes.

## v0.5 - Export and release pack

- Export preset examples.
- Release checklist.
- Itch.io page template.
- Steam page draft template.
- Build folder conventions.

## Later ideas

- Save system example.
- Settings menu.
- Gamepad support.
- Localization example.
- Accessibility notes.
- Optional addon integration examples.
