# Post Merge Tasks

After merging this PR, do the following.

## Repository settings

- Confirm the final repository name.
- Confirm template repository mode is enabled.
- Add repository topics.
- Add a short repository description.

## Local validation

- Clone the repository.
- Open it in Godot.
- Run the main scene.
- Complete `docs/PLAYTEST_CHECKLIST.md`.

## Next technical PR

Recommended PR:

```text
fix: validate and polish demo scene
```

Scope:

- fix broken Godot references;
- improve first room if needed;
- update changelog;
- avoid adding new large systems.

## Release preparation

After validation, prepare tag:

```text
v0.1.0
```
