# Smoke Test

Use this quick smoke test after any technical change.

## Project smoke test

- [ ] Open Godot project.
- [ ] Run main scene.
- [ ] Move forward, backward, left and right.
- [ ] Move camera with mouse.
- [ ] Toggle flashlight.
- [ ] Interact with one object.
- [ ] Confirm UI appears.
- [ ] Exit cleanly.

## Pass condition

The template passes if a new user can start the scene and understand the basic interaction loop.

## Fail condition

The template fails if:

- project does not open;
- main scene does not run;
- player cannot move;
- required scripts are missing;
- interaction system crashes.
