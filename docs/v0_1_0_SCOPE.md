# v0.1.0 Scope

The goal of `v0.1.0` is to provide a small, documented, runnable Godot template.

## Must have

- Godot project opens locally.
- Main scene exists.
- Player can move.
- Basic interaction works.
- Flashlight exists or is documented as planned.
- Documentation entry points exist.
- AI-agent workflow exists.
- Asset/license rules exist.
- Release checklist exists.

## Should have

- One small demo room.
- One readable note.
- One door or switch.
- One objective update.
- One small horror event.

## Not required for v0.1.0

- Full settings menu.
- Save system.
- Inventory grid.
- Enemy AI.
- Steam integration.
- Multiplayer.
- Advanced shaders.
- Final art assets.

## Exit criteria

`v0.1.0` can be tagged after local Godot validation and one complete manual playtest.
