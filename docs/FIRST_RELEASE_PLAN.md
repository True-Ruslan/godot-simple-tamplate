# First Release Plan

Target release: `v0.1.0`.

## Purpose

Publish the first stable template snapshot after local Godot validation.

## Required before release

- Complete local validation.
- Complete playtest checklist.
- Confirm README is accurate.
- Confirm license and credits.
- Confirm no temporary files are committed.
- Confirm repository name and topics.

## Release content

Use `docs/RELEASE_NOTES_v0.1.0.md` as the basis for GitHub release notes.

## Suggested release title

```text
v0.1.0 - Initial Godot Atmospheric Horror Template
```

## Suggested announcement

```text
First public template snapshot for short first-person atmospheric horror projects in Godot.
```

## After release

Start `v0.2.0` planning around a more complete micro horror demo.
