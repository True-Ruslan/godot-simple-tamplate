# v0.2 Ideas

After `v0.1.0`, focus on improving the playable example.

## Candidate features

- complete micro horror demo flow;
- light flicker event;
- simple audio manager example;
- optional VHS shader example;
- settings menu draft;
- export preset example;
- Markdown link check CI;
- Godot headless check CI.

## Recommended v0.2 scope

Choose only three:

1. complete demo flow;
2. one atmosphere event;
3. basic CI structure check.

## Avoid for v0.2

- enemy AI;
- inventory grid;
- save system;
- full game scenario;
- bundled third-party art packs.
